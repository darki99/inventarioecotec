# Sistema Web - Inventario y Ventas

## Descripción
Sistema web desarrollado en PHP y MySQL que permite gestionar productos y registrar ventas aplicando arquitectura por capas.

## Requisitos
- PHP 8+
- MySQL
- XAMPP o servidor local

## Instalación
1. Clonar repositorio
2. Importar database/inventario.sql
3. Configurar credenciales en config/database.php
4. Ejecutar desde /public/productos.php

## Base de Datos
Incluye script SQL en carpeta /database
